Urban Sound Classification Dataset

Download Dataset

The Urban Sound Classification Dataset is part of the UrbanSound8K dataset. It is used for machine learning tasks such as audio classification and urban sound recognition. The dataset provides metadata in a CSV file named UrbanSound8K.csv, containing the following columns:

Column Name Description Example
slice_file_name Name of the audio file slice, including the .wav extension. 100032-3-0-0.wav
fsID Unique identifier for the Freesound.org recording. 100032
start Start time (in seconds) of the audio slice. 0.0
end End time (in seconds) of the audio slice. 0.317551
salience Foreground (1) or background (2) sound indicator. 1
fold Cross-validation fold (1-10) for reproducible experiments. 5
classID Numeric identifier for the sound class. 3
class Sound class label. dog_bark

This dataset is useful for training machine learning models to recognize sounds such as dog barks, car horns, and children playing in urban environments.
