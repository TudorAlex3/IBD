The ImageNet Dataset is one of the largest and most widely-used datasets in computer vision. It provides millions of labeled images for training deep learning models, organized into a hierarchical structure following the WordNet lexical database. The associated metadata is available in a CSV file named imagenet.csv, which contains the following columns:

Column Name Description Example
synid Unique WordNet ID (synset ID) associated with a concept class. n00004475
class_name Name of the class/concept associated with the synset ID. organism
urls Number of image URLs available for this class. 8
flickr_urls Number of Flickr image URLs available for this class. 6

This dataset is a foundational resource for image classification, object detection, and semantic segmentation tasks. It includes a variety of class types such as organisms, artifacts, plants, and more. Examples from the dataset include:

  • SynID: n00015388, Class: animal
  • SynID: n00326094, Class: rock climbing
  • SynID: n00017222, Class: plant
