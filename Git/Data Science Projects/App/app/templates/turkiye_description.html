Dataset Overview

This dataset contains the evaluations of students' performance in a variety of courses in Turkey. The dataset includes information on student attendance, course difficulty, and evaluations of different aspects of the course.

Explanation

The Turkiye Student Evaluation dataset contains survey results from students evaluating their courses and instructors. The dataset has 1,000 rows and 28 columns, including the instructor, class number, attendance, course difficulty, and evaluations for 28 different aspects of the course.
