Chicago Crime Dataset Scatter Plot Analysis

Data Analysis Summary

This scatter plot analysis focuses on key numerical columns from the Chicago Crime dataset. The analysis examines the relationships between geographical coordinates and crime data across different years.

Metric Description Key Observations Outliers (Count)

Outlier Details

Key insights from the scatter plots include:

  • Latitude: Clusters around central Chicago indicate high crime areas, while outliers in suburban regions signify isolated incidents.
  • Longitude: Similar to latitude, clusters around the central area and notable outliers in suburban or less densely populated areas are observed.
  • Year: Trends show how crime distributions shift over the years, with certain years having distinct patterns in crime locations.
