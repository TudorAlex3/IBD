License Number Plate Recognition System Dataset

Download Dataset

The License Number Plate Recognition System Dataset provides comprehensive data on vehicle details, which can be used for building recognition systems or analyzing vehicle-related trends. The dataset is available in a CSV file named license_number.csv and contains the following columns:

Column Name Description Example
model Vehicle model name or code. A1
year Year of manufacture. 2017
price Price of the vehicle (in currency units). 12500
transmission Type of transmission (e.g., Manual or Automatic). Manual
mileage Mileage of the vehicle (in miles or kilometers). 15735
fuelType Type of fuel used (e.g., Petrol, Diesel). Petrol
tax Tax amount (in currency units). 150.0
mpg Miles per gallon (fuel efficiency). 55.4
engineSize Engine size of the vehicle (in liters). 1.4
reference Reference or unique identifier for the vehicle. -

This dataset is highly suitable for machine learning models aimed at license plate recognition, vehicle pricing predictions, or building automated vehicle identification systems.
