Chicago Crime Dataset

Download Dataset

The Chicago Crime Dataset provides detailed information about crimes reported in the city of Chicago. It includes records of incidents categorized by date, type, and location, among other attributes. The associated CSV file, Chicago_Crimes.csv, contains the following columns:

Column Name Description Example
ID Unique identifier for the crime report. 3
Case Number Official case number assigned by the Chicago Police Department. HZ250496
Date Date and time the incident occurred. 05/03/2016 11:40:00 PM
Primary Type Main category of the crime. BATTERY
Description Detailed description of the crime. DOMESTIC BATTERY SIMPLE
Location Description Type of location where the crime occurred. APARTMENT
Arrest Indicates if an arrest was made (True or False). True
Domestic Indicates if the crime was domestic (True or False). True
Latitude Latitude coordinate of the crime location. 41.864073157
Longitude Longitude coordinate of the crime location. -87.706818608
Year Year the incident occurred. 2016

This dataset can be used to analyze trends, identify high-crime areas, and study the effectiveness of law enforcement strategies over time.
