Dataset Overview

The Boston Housing dataset contains information collected from suburbs of Boston, Massachusetts, and is commonly used for regression tasks. The dataset contains 506 rows and 14 columns, with each row representing a housing unit in a particular suburb. The target variable in this dataset is the median value of owner-occupied homes, indicated by the column 'MEDV'.

Columns

  • CRIM: Crime rate per capita in the town.
  • ZN: Proportion of residential land zoned for large lots.
  • INDUS: Proportion of non-retail business acres per town.
  • CHAS: Charles River dummy variable (1 if tract bounds river, 0 otherwise).
  • NOX: Nitrogen oxide concentration (parts per 10 million).
  • RM: Average number of rooms per dwelling.
  • AGE: Proportion of owner-occupied units built before 1940.
  • DIS: Weighted distance to employment centers.
  • RAD: Index of accessibility to radial highways.
  • TAX: Property tax rate per $10,000.
  • PTRATIO: Pupil-teacher ratio by town.
  • B: Proportion of residents of African American descent.
  • LSTAT: Percentage of lower status population.
  • MEDV: Median value of owner-occupied homes (target variable).

Explanation

The Boston Housing dataset is often used in regression analysis to predict the value of homes in various Boston suburbs based on various features such as crime rate, proximity to employment centers, and other socio-economic factors. It consists of 506 samples with 13 features and one target variable, 'MEDV', which indicates the median value of homes in $1,000s.

Target variable: 'MEDV' is the continuous value that we aim to predict based on the features provided.
