Turkiye Student Evaluation Dataset Description

Dataset Information

The Turkiye Student Evaluation dataset consists of data collected to evaluate the teaching performance of instructors in different courses. The dataset contains multiple rows, each representing a student's response to evaluation questions, and columns detailing the student's class, attendance, difficulty of the course, and their responses to 28 survey questions.

Data Description

Columns:
1. "instr"       - Instructor identifier (categorical)
2. "class"       - Class identifier (categorical)
3. "nb.repeat"   - Number of times the student has taken the course (numerical)
4. "attendance"  - Attendance level of the student (numerical: 0=low, 4=high)
5. "difficulty"  - Difficulty level of the course as perceived by the student (numerical: 1=easy, 5=difficult)
6. "Q1" - "Q28"  - Responses to evaluation questions (numerical: 1=very poor, 5=excellent)

Key Features

  • Instructor and Class: Allows analysis of performance trends per instructor and class.
  • Attendance: Correlation between attendance levels and evaluations can be studied.
  • Difficulty: Relationship between perceived course difficulty and evaluation scores.
  • Survey Responses: 28 evaluation questions designed to assess teaching quality.

Handling Missing Values

If missing values are encountered, they can be handled as follows:

  • For numerical columns such as "nb.repeat," "attendance," and "difficulty," fill missing values with the median or mode.
  • For evaluation responses (Q1-Q28), consider filling with the mean score for each question or dropping rows with too many missing values.

Key Insights

The dataset can be utilized to:

  • Analyze instructor performance across multiple courses.
  • Identify patterns in student feedback for improvement.
  • Correlate attendance and difficulty ratings with evaluation scores.
  • Predict overall teaching performance using machine learning models.
