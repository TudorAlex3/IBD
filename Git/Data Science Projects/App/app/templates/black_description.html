Dataset Overview

The Black Friday dataset contains information on customer purchases during a Black Friday sale. It includes demographic details of customers, product details, and purchase amount, allowing for analysis of purchasing behavior and trends.

Columns

  • User_ID: Unique identifier for each user.
  • Product_ID: Unique identifier for each product.
  • Gender: Gender of the user (M/F).
  • Age: Age group of the user.
  • Occupation: Occupation code of the user.
  • City_Category: Category of the city (A, B, C).
  • Stay_In_Current_City_Years: Number of years the user has stayed in the current city.
  • Marital_Status: Marital status of the user (0 = Single, 1 = Married).
  • Product_Category_1: Main product category.
  • Product_Category_2: Secondary product category (if applicable).
  • Product_Category_3: Tertiary product category (if applicable).
  • Purchase: Purchase amount.

Explanation

The dataset can be used for various analyses, including:

  • Customer demographics and purchase behavior.
  • Product performance analysis.
  • Segmentation of customers based on purchasing trends.
  • Prediction of purchase amounts using machine learning techniques.
