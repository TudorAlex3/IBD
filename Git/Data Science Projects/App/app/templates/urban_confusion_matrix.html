Confusion Matrix Explanation

Explanation

The confusion matrix visually represents how well a classification model is performing by comparing predicted values with actual values. It consists of:

  • Diagonal entries: These represent correctly classified instances for each sound category (e.g., Silence, Noise, Music).
  • Off-diagonal entries: These represent misclassifications, where a model predicts one category but the true label belongs to a different category.
  • Random Forest: Exhibits the most accurate predictions, with the least number of misclassifications.

Conclusion: Random Forest provides the highest accuracy and is the most reliable model for the urban sound classification dataset. Logistic Regression and Decision Tree also perform well but have slightly higher misclassification rates.
