The Market Basket Analysis Dataset provides a rich dataset for exploring customer purchase patterns and basket trends. It contains transactional data, including items purchased, quantities, and customer details. The dataset is structured in a CSV file named market.csv with the following columns:

Column Name Description Example
BillNo Unique invoice number for the transaction. 536365
Itemname Name of the purchased item. WHITE HANGING HEART T-LIGHT HOLDER
Quantity Number of units purchased. 6
Date Timestamp of the transaction. 01.12.2010 08:26
Price Unit price of the item (in GBP). 2.55
CustomerID Unique identifier for the customer. 17850
Country Country where the purchase was made. United Kingdom

This dataset is ideal for exploring customer segmentation, association rule mining, and predictive analytics in retail. By analyzing this data, businesses can identify frequent item sets and improve marketing strategies to boost sales.
