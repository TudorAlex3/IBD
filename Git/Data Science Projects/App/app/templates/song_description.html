Dataset Overview

The Million Song Dataset is a freely available collection of audio features and metadata for a million contemporary popular music tracks. It is designed to provide a comprehensive resource for music information retrieval and related fields. The dataset contains features extracted from audio tracks, including year, tempo, loudness, and more, making it suitable for analysis and modeling tasks in musicology and machine learning.

Columns

  • TC01, TC02...: Additional derived features representing track characteristics for modeling and analysis.

Explanation

This dataset contains data points about songs, each represented by a set of features. Key columns include:

  • Year: Indicates the year of release for the music track, useful for trend analysis over time.
  • TA01-TA12: Feature vectors capturing time-aligned audio properties of the tracks.
  • TC01-TC78: Track-level feature columns derived from audio analysis tools.

The dataset supports applications in music recommendation systems, genre classification, trend analysis, and machine learning tasks like regression and clustering.
