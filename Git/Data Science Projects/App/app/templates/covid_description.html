COVID-19 Prediction Dataset

Download Dataset

The COVID-19 Prediction Dataset provides data on COVID-19 cases reported across the globe. It includes information about confirmed cases, deaths, and recoveries for various countries and regions, along with geographical coordinates and dates. The associated CSV file, covid.csv, contains the following columns:

Column Name Description Example
Province/State Name of the state or province (if available). California
Country/Region Name of the country or region. United States
Lat Latitude of the location. 33.0
Long Longitude of the location. -65.0
Date Date of the record in MM/DD/YY format. 1/22/20
Confirmed Number of confirmed cases. 100
Deaths Number of deaths reported. 2
Recovered Number of recoveries reported. 5

This dataset is essential for tracking the global spread of COVID-19 and can be used for predictive modeling, trend analysis, and forecasting the progression of the pandemic in specific regions.
