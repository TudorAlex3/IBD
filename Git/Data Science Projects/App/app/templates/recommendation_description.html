Recommendation Engine Dataset

Download Dataset

The Recommendation Engine Dataset contains data for building a recommendation engine, particularly for books. This dataset includes book details such as ISBN, title, author, year of publication, publisher, and URLs for book cover images. The associated CSV file, output_first_12000.csv, contains the following columns:

Column Name Description Example
ISBN Unique identifier for the book. 0195153448
Book-Title Title of the book. Classical Mythology
Book-Author Author of the book. Mark P. O. Morford
Year Year of publication. 2002
Publisher Name of the publisher. Oxford University Press
Image-URL-S Small image URL for the book cover. View
Image-URL-M Medium image URL for the book cover. View
Image-URL-L Large image URL for the book cover. View

This dataset can be used for building recommendation engines for books, analyzing popular authors or publishers, and visualizing book details using cover images.
